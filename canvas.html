Start Using Canvas

With Canvas, you can incorporate data.world into your courses with Canvas's world-class LMS.

Create an Organization on data.world

To make sure you can invite all your students to upload and manage data, you'll first want to create an organization for your class on data.world:

  1. Click here to enter details and create your organization
  2. Invite members
    Note: You can easily invite members via link instead of manually entering each email address

*If you've already created an org on data.world, you can enable invite links through your org's Settings > Preferences:

  1. Check Allow users to request to join this organization

  2. Go to the People tab of your organization and copy the link to invite members.

Sharing this link will prompt users to create an account if necessary and request an invite to the org. You will need to accept any requests to join your org.

Install the data.world LTI in Canvas

Once you've created your classroom org on data.world, you can install data.world's LTI into your Canvas course:

  1. Navigate to the desired course in Canvas and access the course settings menu
  2. Within the course settings menu, access the course "Apps" menu
  3. Select the Add App button, (+ App), from within the course apps submenu. This will cause an application configuration modal to pop up
  4. Use the Config Link Generator form to create your configuration link
  5. Fill out the configuration settings in the configuration modal:
    • Configuration Type: By URL
    • Name: data.world
    • Consumer Key: Leave Blank
    • Shared Secret: Leave Blank
    • Configuration URL: Enter link from the Config Link Generator Form
