Start Using KNOTS

With KNOTS, you can quickly and easily leverage ETL without complex manual configurations.

NOTE: KNOTS is currently only available on MacOS

Download KNOTS & Docker

Before you can start replicating your data, you'll need to download and install KNOTS and Docker:

  1. Download the .dmg from the KNOTS repository on GitHub (KNOTS is built on top on Singer.io taps & targets)
  2. Download Docker NOTE: You'll need to create a free Docker account before downloading
  3. Install KNOTS and Docker on your computer

Start Your KNOT

Now that KNOTS & Docker are installed, you can begin creating your first KNOT:

  1. Open KNOTS & Docker (make sure Docker is running by checking the application in the top bar of your computer)
  2. Select your tap (the source you want to use to replicate data). NOTE: Some taps may require additional accounts or installed apps.
  3. Configure your tap. You'll need to enter the following information. NOTE: Some taps may require additional information
    • Hostname
      NOTE: If you are running your database locally, enter host.docker.internal instead of localhost
    • Port
      NOTE: Most apps have a default port (5432 for PostgreSQL, 3306 for MySQL).
    • Database name
    • Username & Password for your database

Choose Replication Options

Choose the data you'd like to replicate by checking boxes in the list of available data.

Choose Your Target

Choose data.world from the list of targets to replicate your data on data.world

Configure Your Target

Configure your target. You'll need to enter the following information:

  1. Dataset URL
    Enter the URL for the dataset you'd like replicate to, or use the Browse button and sign into data.world to browse through a list of options.
  2. API Key
    You can find your API Key in your data.world advanced settings here.

Save & Run

Once you configure your target, you can save & run your KNOT. NOTE: Some replication may take a while if the datasets are large, but don't worry! You can pause and restart your KNOT at any time without any data loss.

What Else Can I Do?

Once you've completed a KNOT, you can always take actions on your completed KNOTS, like:

  • Sync new data
  • Sync all data
  • Edit your KNOT
  • Export your KNOT
    NOTE: Exporting KNOTS will allow you to use the job scheduler of your choice to make sure your data is always up to date.
  • Delete your KNOT
  • Create a new KNOT
